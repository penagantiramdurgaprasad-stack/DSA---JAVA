# Sliding Window - DSA (Java)
This folder contains Sliding Window based data structures and algorithm problems commonly asked in interviews.

# When to Use
- Fixed Window
- Variable Window
- Input is Array/String
- SubArray/SubString required
- Range in Contiguous
- Asked for Min/Max/Count/length

# Problem List
  Fixed Window
- Maximum SubArray of size K
- Average of SubArray of Size k
- First negative Number in every window
- Count Occurrence of anagram

  Variable Window

- Longest SubString Without Repeating Characters
- Longest Substring without K distinct Characters
- Minimum window Substring
- Fruits into Basket
- SubArray Product less than k

  Advanced

- Longest Repeating Character Replacement
- Binary SubArrays with Sum
- Count Number of Nice SubArrays
- SubArrays with k Different Integers
- Smallest Window Containing All Characters
